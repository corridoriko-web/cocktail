# 鸡尾酒配方搜索网站

一个简洁美观的鸡尾酒配方搜索网站，支持前后台管理。

## 功能特性

### 前台功能（公开访问）
- 鸡尾酒配方搜索（支持名称、配料、标签搜索）
- 卡片式展示，点击查看详情
- 响应式设计，支持移动端
- 无需登录，任何人都可以访问

### 后台管理（密码保护）
- 添加新的鸡尾酒配方
- 编辑现有配方
- 删除配方
- 批量文字替换功能
- 需要输入管理密码才能访问

## 安装运行

### 1. 安装依赖

```bash
npm install
```

### 2. 构建前端

```bash
npm run build
```

### 3. 启动服务器

```bash
npm start
# 或直接运行
node server/index.js
```

服务将运行在 http://localhost:3000

## 使用说明

### 访问网站
- **前台首页**：http://localhost:3000/
- **管理后台**：http://localhost:3000/#/admin

### 管理员登录
- 管理密码：`Jump123456`
- 登录后可以进行添加、编辑、删除配方等操作

### 添加配方
1. 访问管理后台 `/admin`
2. 输入管理密码登录
3. 点击"添加配方"标签
4. 填写配方信息并保存

### 批量替换文字
1. 在管理后台点击"批量替换"标签
2. 输入要查找的文字和替换后的文字
3. 点击"执行替换"

## 技术栈

- 前端：Vue 3 + Vite
- 后端：Express.js
- 数据存储：JSON文件

## 数据结构

每个鸡尾酒配方包含：
- `name`: 名称
- `name_en`: 英文名称
- `ingredients`: 配料（每行一个）
- `steps`: 制作步骤（每行一个）
- `description`: 描述
- `image`: 图片URL
- `tags`: 标签（逗号分隔）

## 注意事项

- 数据存储在 `data/cocktails.json` 文件中
- 图片需要提供外部 URL
- 管理密码硬编码在前端，适合个人或小团队使用
